# Color Palette
## SmartLibrary AI - IoT Based Smart Library Seat Management System

This document specifies the official color palette design token system for SmartLibrary AI, complete with HSL and HEX configurations mapping to Light and Dark modes.

---

### 1. Brand Color Palettes

#### 1.1 Primary Accent (Indigo / Royal Blue)
Provides the core brand indicator for highlights, selected buttons, active navigation, and brand elements.
*   **Indigo 500 (Primary):** `#6366f1` / `hsl(239, 84%, 66%)`
*   **Indigo 600 (Primary Hover):** `#4f46e5` / `hsl(243, 75%, 59%)`
*   **Indigo 50 (Soft Background Accent):** `#eef2ff` / `hsl(226, 100%, 97%)`

#### 1.2 Neutral Scales (Slate)
Handles typography text weights, card backgrounds, borders, and sidebar divisions.
*   **Light Mode Neutral backgrounds:**
    *   *App Surface Background:* `#f8fafc` / `hsl(210, 40%, 98%)` (Slate 50)
    *   *Card Surface:* `#ffffff` / `hsl(0, 0%, 100%)`
    *   *Muted Border:* `#e2e8f0` / `hsl(214, 32%, 91%)` (Slate 200)
*   **Dark Mode Neutral backgrounds:**
    *   *App Surface Background:* `#0f172a` / `hsl(222, 47%, 11%)` (Slate 900)
    *   *Card / Dialog Surface:* `#1e293b` / `hsl(217, 33%, 17%)` (Slate 800)
    *   *Muted Border:* `#334155` / `hsl(215, 25%, 27%)` (Slate 700)
